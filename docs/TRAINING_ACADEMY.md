# 培训学堂 — 产品说明文档

## 概述

EM-BOX 培训学堂是一套面向企业招聘团队的 AI 驱动培训管理系统。通过将面试中发现的候选人薄弱点与结构化培训课程自动匹配，帮助招聘团队系统性地提升候选人通过率。

**核心价值**：面试薄弱点分析 → 智能推荐课程 → 针对性培训 → 提升面试通过率

---

## 功能清单

### 1. 课程管理（课程管理 Tab）

| 功能 | 说明 |
|------|------|
| 创建课程 | 设置课程标题、分类维度（沟通表达/专业能力/应变能力/综合素质）、难度等级（初级/中级/高级）、时长、课程描述 |
| 课程章节 | 支持文字、视频、链接三种内容类型，每个章节可独立编辑排序 |
| 本地上传 | 章节内容支持本地上传文件（文档/视频/图片），自动上传至云存储并填入 URL |
| 参考资料 | 可为课程添加参考资料（PDF/视频/文章/练习），支持本地上传附件 |
| 考核配置 | 设置考核方式（测验/AI评审/人工评审）和及格分数线 |
| 编辑/删除 | 管理员和招聘运营可编辑课程，仅管理员可删除 |

### 2. 学习路径（学习路径 Tab）

| 功能 | 说明 |
|------|------|
| 创建路径 | 将多门课程组合成结构化学习路径，设定顺序和难度等级 |
| 课程排序 | 拖拽调整课程顺序，标注必修/选修 |
| 认证路径 | 标记为认证路径，学员完成后可颁发证书（预留功能） |
| 报名管理 | 点击路径卡片「X 人已报名」进入报名管理弹窗 |
| 报名进度 | 查看每位学员的学习状态（已报名/学习中/已完成/未通过）和进度百分比 |
| 状态更新 | 直接在下拉菜单中更新学员状态，拖动滑块调整进度 |
| 取消报名 | 支持取消学员报名 |
| 批量报名 | 选择目标课程或路径，搜索并多选候选人，一键批量报名 |

### 3. 培训记录（培训记录 Tab）

| 功能 | 说明 |
|------|------|
| 报名列表 | 查看所有培训报名记录，支持按状态筛选（全部/已报名/学习中/已完成/未通过） |
| 进度追踪 | 显示每门课程的学习进度百分比 |
| 考核录入 | 对进行中的培训录入考核分数（0-100分） |
| 分数对比 | 显示培训前面试分 → 考核分的对比 |
| CSV 导出 | 导出培训记录为 CSV 文件（含中文表头，兼容 Excel） |

### 4. 薄弱分析（薄弱分析 Tab）

| 功能 | 说明 |
|------|------|
| 面试薄弱维度 | 基于面试不通过记录，自动统计高频薄弱维度排名 |
| 受影响候选人 | 每个薄弱维度下展示受影响的候选人名单 |
| 关联课程 | 显示每个薄弱维度可匹配的培训课程数量 |
| 智能推荐 | 输入候选人 ID 和姓名，AI 自动分析薄弱维度并推荐匹配课程 |
| 一键报名 | 推荐结果中可直接将候选人报名到推荐课程 |

### 5. 效果统计（效果统计 Tab）

| 功能 | 说明 |
|------|------|
| 培训总览 | 展示培训完成人次、平均分数提升、提升率 |
| 分维度效果 | 按分类维度展示培训前后的平均分对比 |
| 提升幅度 | 每个维度的具体提升分数和提升比例 |

### 6. 权限体系

| 角色 | 培训权限 |
|------|---------|
| 管理员 (admin) | 查看 + 管理全部培训功能，可删除课程和路径 |
| 招聘运营 (recruiter) | 查看 + 管理培训功能，可创建课程/路径、管理报名 |
| 用人经理 (hiring_manager) | 仅查看培训内容 |
| 访客 (viewer) | 仅查看培训内容 |

## 使用说明

### 创建第一门课程

1. 进入「培训学堂」→「课程管理」Tab
2. 点击右上角「新建课程」按钮
3. 填写课程信息：
   - **课程标题**（必填）：课程名称
   - **分类维度**：选择与课程最相关的面试维度
   - **难度等级**：初级/中级/高级
   - **时长**：预计完成时间（分钟）
   - **课程描述**：简要说明课程内容和目标
   - **胜任力维度**（可选）：关联到特定的岗位胜任力模型
4. 展开「课程章节内容」添加章节：
   - 点击「+ 添加章节」
   - 输入章节标题
   - 选择内容类型（文字/视频/链接）
   - 填写内容或点击「本地上传」上传文件
5. 添加参考资料（可选）：
   - 点击「+ 添加参考资料」
   - 输入标题、选择类型、填入 URL 或点击「上传」按钮上传文件
6. 设置考核配置：
   - 选择考核方式（测验/AI评审/人工评审）
   - 设置及格分数（默认 60 分）
7. 点击「创建课程」

### 创建学习路径

1. 进入「培训学堂」→「学习路径」Tab
2. 点击右上角「新建学习路径」
3. 填写路径信息：
   - **路径名称**：如「前端开发工程师入职培训」
   - **分类**和**难度等级**
   - **路径描述**：说明该路径的目标和适用人群
   - 可选：勾选「认证路径」
4. 在「包含课程」区域，从可用课程列表中点击添加课程
5. 使用上下箭头调整课程学习顺序
6. 点击「创建路径」或「保存修改」

### 管理路径报名

1. 在学习路径卡片底部，点击「X 人已报名」
2. 在弹出窗口中：
   - **添加候选人**：搜索候选人姓名 → 点击「报名」
   - **更新状态**：在下拉菜单中修改学员状态
   - **更新进度**：拖动滑块调整学习进度（0-100%）
   - **取消报名**：点击「取消报名」移除学员
3. 顶部显示总报名人数、已完成人数、进行中人数

### 批量报名

1. 在课程管理或学习路径 Tab，点击「批量报名」
2. 选择报名目标类型（课程/学习路径）
3. 在下拉菜单中选择具体课程或路径
4. 搜索候选人并勾选多选
5. 已选候选人显示为标签，可点击 X 移除
6. 点击「报名 N 人」提交
7. 查看结果：成功报名 X 人，跳过 Y 人（含跳过原因）

### 录入考核分数

1. 进入「培训记录」Tab
2. 找到状态为「已报名」或「学习中」的学员
3. 点击「录入考核」按钮
4. 输入 0-100 的分数
5. 点击「确认」提交
6. 系统自动判断通过/未通过（≥60 分为通过）

### 使用薄弱分析推荐课程

1. 进入「薄弱分析」Tab
2. 查看系统自动统计的高频薄弱维度
3. 在「智能推荐课程」区域：
   - 输入候选人 ID
   - 输入候选人姓名
   - 点击「推荐课程」
4. 系统基于该候选人的面试薄弱维度，自动匹配相关课程
5. 点击「报名」将候选人加入推荐课程

### 上传文件到课程

**支持的格式**：
- 文档：PDF, DOC, DOCX, PPT, PPTX, XLS, XLSX, TXT, MD
- 视频：MP4, MOV, WebM, AVI, MKV
- 图片：JPG, JPEG, PNG, GIF
- 压缩包：ZIP, RAR
- 单文件最大：500MB

**上传方式**：
- 课程章节中：点击内容输入框旁的「本地上传」按钮
- 参考资料中：点击 URL 输入框旁的「上传」按钮
- 上传后 URL 自动填入输入框

## 技术架构

| 层级 | 技术 |
|------|------|
| 前端 | React 19 + TypeScript + Tailwind CSS + Motion |
| 开发后端 | Express 4 + PostgreSQL |
| 生产后端 | Supabase Edge Functions (Deno) |
| 数据库 | PostgreSQL (Supabase) |
| 文件存储 | Supabase Storage |
| AI 引擎 | 多模型 LLM (OpenAI/Anthropic/Gemini/DeepSeek/Zhipu 等) |
| 部署 | Vercel (前端) + Supabase (后端) |

## 数据表

| 表名 | 说明 |
|------|------|
| `training_courses` | 培训课程（含章节内容 JSONB、参考资料 JSONB、考核配置 JSONB） |
| `training_enrollments` | 课程报名记录（含状态、进度、分数） |
| `training_assessments` | 考核评估记录 |
| `training_paths` | 学习路径定义 |
| `training_path_courses` | 路径-课程关联（含排序、必修标记） |
| `training_path_enrollments` | 路径报名记录（含状态、进度） |
