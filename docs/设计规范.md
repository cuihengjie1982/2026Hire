# EM-BOX AI招聘平台 设计规范

## 1. 设计系统概述

### 1.1 设计理念

本设计系统遵循「专业高效、清晰可辨」的原则，采用低饱和度的蓝色系作为主色调，配合明亮的功能色，在保证后台管理效率的同时，提供现代化的视觉体验。

### 1.2 设计工具

- 框架：React 19 + TypeScript + Vite
- 样式：Tailwind CSS 4.x
- 图标：Lucide React
- 动画：Motion React
- 图表：Recharts

## 7. 动画规范

### 7.1 动画库

使用 `motion/react` 实现动画效果。

### 7.2 通用动画

| 动画名称 | 配置 | 用途 |
|---------|------|------|
| 淡入 | `{ opacity: 0 → 1, duration: 0.2s }` | 页面切换 |
| 滑入 | `{ opacity: 0, y: 10 → y: 0, duration: 0.3s }` | 模态框 |
| 缩放 | `{ scale: 0.95 → 1, duration: 0.2s }` | 弹窗 |

### 7.3 交互动画

| 场景 | 动画 |
|-----|------|
| 按钮悬停 | 背景色变深，transition-colors 150ms |
| 列表项悬停 | bg-gray-50/50, transition-colors |
| 开关切换 | spring动画，stiffness: 500, damping: 30 |
| 下拉展开 | opacity + y 联动，duration 0.2s |

### 7.4 特殊效果

| 效果 | 配置 |
|-----|------|
| 录制中指示灯 | animate-pulse |
| 加载中 | animate-spin |
| 倒计时数字 | text-5xl font-extrabold tracking-widest |

## 8. 组件规范

### 8.1 按钮

**主要按钮**
```tsx
<button className="bg-[#1a4bc4] hover:bg-[#0c2b7a] text-white px-4 py-2 rounded-lg text-sm font-medium transition-colors">
```

**次要按钮**
```tsx
<button className="border border-gray-300 hover:bg-gray-50 text-gray-700 px-4 py-2 rounded-lg text-sm font-medium transition-colors">
```

**危险按钮**
```tsx
<button className="bg-red-500 hover:bg-red-600 text-white px-4 py-2 rounded-lg text-sm font-bold">
```

### 8.2 表格

```tsx
<div className="bg-white rounded-xl border border-gray-200 overflow-hidden">
  <table className="w-full text-left border-collapse">
    <thead className="bg-gray-50 border-b border-gray-200">
      <tr>
        <th className="py-3 px-6 font-medium text-gray-500 text-sm">表头</th>
      </tr>
    </thead>
    <tbody className="divide-y divide-gray-100">
      <tr className="hover:bg-gray-50/50 transition-colors">
        <td className="py-4 px-6">内容</td>
      </tr>
    </tbody>
  </table>
</div>
```

### 8.3 表单输入

**文本输入框**
```tsx
<input
  type="text"
  className="w-full bg-gray-50 border border-gray-200 rounded-lg px-3 py-2 text-sm focus:outline-none focus:ring-2 focus:ring-[#1a4bc4]/20 focus:border-[#1a4bc4]"
/>
```

**下拉选择框**
```tsx
<div className="relative">
  <select className="appearance-none w-full border border-gray-200 rounded-lg py-2 pl-4 pr-10 outline-none">
    <option>选项</option>
  </select>
  <ChevronDown className="absolute right-3 top-1/2 -translate-y-1/2 w-4 h-4 text-gray-400" />
</div>
```

**开关**
```tsx
<button className="w-11 h-6 rounded-full relative transition-colors bg-[#1a4bc4]">
  <motion.div layout className="absolute top-1 left-1 bg-white w-4 h-4 rounded-full shadow-sm" />
</button>
```

### 8.4 徽章/标签

**状态徽章**
```tsx
<span className="inline-flex items-center px-3 py-1 rounded-full text-xs font-bold bg-emerald-100 text-emerald-700">
```

**计数徽章**
```tsx
<span className="w-6 h-6 text-xs bg-[#1a4bc4] text-white rounded-full flex items-center justify-center">
```

### 8.5 卡片

**统计卡片**
```tsx
<div className="bg-[#E6F4EA] rounded-xl p-5 border border-[#108E11]/10">
  <div className="flex items-center text-green-700 font-medium text-sm mb-2">
    <Icon className="w-4 h-4 mr-1.5" /> 标题
  </div>
  <div className="text-2xl font-bold text-green-800">数值</div>
</div>
```

### 8.6 模态框

**基础结构**
```tsx
<div className="fixed inset-0 z-50 flex items-center justify-center p-4 bg-gray-500/20 backdrop-blur-sm">
  <motion.div
    initial={{ opacity: 0, scale: 0.95, y: 10 }}
    animate={{ opacity: 1, scale: 1, y: 0 }}
    className="bg-white w-full max-w-4xl rounded-2xl shadow-xl"
  >
    {/* 内容 */}
  </motion.div>
</div>
```

## 13. 设计检查清单

新页面/组件开发时需确认：

- [ ] 色彩系统是否正确使用（主色、功能色、状态色）
- [ ] 字体规范是否遵循（字号、字重）
- [ ] 间距是否一致（4px倍数）
- [ ] 圆角是否规范
- [ ] 阴影是否适度
- [ ] 动画是否流畅（无闪烁、符合规范）
- [ ] 交互状态是否完整（hover、active、disabled）
- [ ] 响应式布局是否适配
- [ ] 无障碍支持（键盘导航、ARIA标签）
